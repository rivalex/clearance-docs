# Дашборд

Дашборд - это стартовая страница панели Clearance. Он даёт беглый обзор состояния авторизации вашего приложения - без необходимости какой-либо настройки.

## Что он показывает

- **Всего ролей** - количество всех ролей Spatie.
- **Всего guard'ов** - количество всех guard'ов, управляемых Clearance (см. [Guards](/ru/features/guards)).
- **Всего разрешений** - количество всех разрешений Spatie.
- **Группы разрешений** - количество различных групп разрешений, определённых по префиксу именования (`group` в `group-action`, напр. `orders` в `orders-create`). См. [Разрешения](/ru/features/permissions) для соглашения об именовании.
- **Контекстные назначения ролей** - количество назначений ролей, привязанных к конкретному экземпляру модели (Store, Tenant, Project). См. [Контекстную авторизацию](/ru/features/contextual-authorization).
- **Топ-5 ролей по количеству пользователей** - пять ролей с наибольшим числом назначенных пользователей, в рейтинге.

## Доступ к Дашборду

| URI | Имя маршрута | Описание |
|---|---|---|
| `/clearance` | `clearance.home` | Дашборд |

Чтение Дашборда требует разрешения `clearance-access`, как и любой другой раздел панели. См. [Super Admin](/ru/features/super-admin#panel-access-fine-grained-write-permissions) для полной таблицы контроля доступа.

## Встраивание компонента

Дашборд - это самодостаточный Livewire-компонент. Подключите его в любом месте ваших собственных layout'ов вместо готового маршрута `/clearance` или в дополнение к нему:

```blade
<livewire:clearance::dashboard />
```
