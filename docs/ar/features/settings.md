# الإعدادات

تُهيّئ لوحة الإعدادات كيفية عرض الأدوار والحراس في باقي أجزاء لوحة Clearance، بالإضافة إلى الدور الافتراضي المُعيَّن للمستخدمين الجدد.

## ما تقوم به اللوحة

- **بيانات العرض الوصفية** لكل دور/حارس - اسم العرض، والوصف، وأيقونة SVG معقّمة + لون سداسي عشري، تُعرض في جميع أنحاء اللوحة.
- **الدور الافتراضي** - الدور المُعيَّن تلقائياً للمستخدمين المسجَّلين حديثاً عندما تكون `auto_assign_default_role` مضبوطة على `true`.
- **التعيين الجماعي للدور الافتراضي** - تعيين بنقرة واحدة للدور الافتراضي الحالي لكل مستخدم موجود لا يمتلكه بعد.
- **إظهار الأيقونات** - تبديل عرض الأيقونات في جميع أنحاء اللوحة.

## بيانات العرض الوصفية لكل دور/حارس

<Screenshot src="/screenshots/settings-role-meta.png" alt="نموذج بيانات عرض الدور الوصفية مع منتقيات الأيقونة واللون" caption="تعديل اسم العرض، والوصف، والأيقونة، واللون لدور ما." />

تمر كل أيقونة SVG يقدّمها المستخدم عبر مُعقِّم بقائمة سماح قبل التخزين ومرة أخرى عند العرض، بحيث يمكن تعديل بيانات العرض الوصفية دون إدخال ترميز غير آمن إلى اللوحة.

## الدور الافتراضي والتعيين التلقائي

```php
// config/clearance.php
'auto_assign_default_role' => true, // requires a default role set in Settings
```

::: tip عيّن الدور الافتراضي قبل تفعيل التعيين التلقائي
لا تأثير لـ `auto_assign_default_role` حتى يُهيَّأ دور افتراضي في لوحة الإعدادات. عيّن الدور الافتراضي أولاً، ثم فعّل المفتاح.
:::

استخدم **التعيين الجماعي** في اللوحة في أي وقت لتعيين الدور الافتراضي الحالي بأثر رجعي للمستخدمين الموجودين الذين لا يمتلكونه بعد - مفيد بعد تغيير الدور الافتراضي، أو بعد تفعيل `auto_assign_default_role` على تطبيق يحتوي على مستخدمين موجودين مسبقاً.

## التحكم بالوصول

يتطلب الوصول للقراءة صلاحية `clearance-access`. يتطلب تعديل بيانات العرض الوصفية، أو الأيقونات، أو الألوان، أو الدور الافتراضي صلاحية `clearance-settings-write`:

```php
$user->givePermissionTo('clearance-access');
$user->givePermissionTo('clearance-settings-write');
```

## المسار والمكوّن المُضمَّن

| URI | اسم المسار | الوصف |
|---|---|---|
| `/clearance/settings` | `clearance.settings` | بيانات العرض الوصفية (الأيقونات، الألوان)، الدور الافتراضي، الإعدادات العامة |

```blade
<livewire:clearance::settings.settings-manager />
```

## قاعدة البيانات

| الجدول | الأعمدة الأساسية |
|---|---|
| `clr_meta` | `subject_type`، `subject_key` (فريد معاً) - `display_name`، `description`، `icon_svg`، `color` |
| `clr_settings` | `key` (فريد)، `value` |
